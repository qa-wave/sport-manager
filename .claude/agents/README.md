# Tým agentů — sport-manager

Multi-tenant SaaS pro sportovní kluby (kalendář, RSVP, attendance, platby). Kompletní tým (17) — technicky, vizuálně i marketingově prodatelný.

| Agent | Zaměření v projektu |
|---|---|
| produktovy-manazer | Vize SaaS, persony klubů, MVP scope, priorizace |
| projektovy-manazer | Orchestrace monorepa (api/web/mobile/workers), fázování |
| business-analytik | User stories pro RSVP/attendance/platby, akceptační kritéria |
| softwarovy-architekt | Multi-tenant architektura, API design, datový model, SSE |
| backend-vyvojar | NestJS/Node API, DB, byznys logika, Stripe integrace, auth |
| frontend-vyvojar | Next.js web app, komponenty, state, Tailwind |
| mobilni-vyvojar | React Native mobilní app (apps/mobile) |
| devops-inzenyr | CI/CD, Docker, Vercel deploy, monitoring, workers |
| security-specialista | Multi-tenant izolace, platby, auth/authz, hardening |
| qa-tester | Playwright test plány, regrese, akceptace |
| ux-designer | Flows RSVP/kalendář/komunikace, IA, wireframy |
| ui-designer | Obrazovky, komponenty, stavy, implementačně připravené UI |
| jack-dorsey-ux | Polishing, ruthless minimalismus, ověření v prohlížeči |
| brand-designer | Brand klubového SaaS — logo, paleta, typografie |
| web-designer | Konverzní landing pro self-service registraci klubů |
| marketer | Positioning, akviziční kanály, launch plán |
| copywriter | UI copy, e-maily (RSVP/platby), microcopy |
