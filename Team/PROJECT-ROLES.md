# Project Roles — sport-manager

> Které agenty volat na jaké úkoly. Group: `web-app-saas`.
> Agenti jsou definováni v [`agents/`](agents/).

## Sada agentů tohoto projektu

- `backend-vyvojar`
- `frontend-vyvojar`
- `softwarovy-architekt`
- `security-specialista`
- `qa-tester`
- `devops-inzenyr`
- `produktovy-manazer`
- `memory-curator`
- `mcp-broker`

## Mapování úkol → agent

| Úkol | Doporučený agent |
|---|---|
| Architektonická rozhodnutí, volba stacku | `softwarovy-architekt` (kde existuje) |
| Implementace frontend (React/Next.js) | `frontend-vyvojar` |
| Implementace backend (API, DB) | `backend-vyvojar` |
| UI/UX design, screens, komponenty | `ui-designer` |
| Visual polishing, design critique | `jack-dorsey-ux` |
| Marketing/landing copy, microcopy | `copywriter` |
| Marketing web design | `web-designer` |
| SEO optimalizace, schema, keyword research | `seo-strategist` |
| Test plány, QA, regression | `qa-tester` |
| Bezpečnostní review, threat model | `security-specialista` |
| Infra, CI/CD, deploy | `devops-inzenyr` |
| Produktová strategie, MVP scope | `produktovy-manazer` |
| Business analýza, user stories, AC | `business-analytik` |
| Project management, orchestrace | `projektovy-manazer` |
| Správa second-brain (compile, dedup, index) | `memory-curator` |
| Volba MCP nástroje pro úkol | `mcp-broker` |

> **Tip**: pro otevřené výzkumné otázky bez konkrétního agenta používej `general-purpose`.
> Pro >3 grep/find napříč kódem používej `Explore`.
